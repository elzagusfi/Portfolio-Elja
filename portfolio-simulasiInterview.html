Interview Simulation Project

Many job seekers struggle with interview preparation, particularly when it comes to practicing responses to common questions under pressure. Without the opportunity to rehearse with mentors or peers, candidates often face difficulties in building the confidence needed for a successful interview.
To address this challenge, I created an Automated Interview Simulation platform that enables users to practice interviews in a realistic and structured format. Users can simply input their name, select the desired duration, and respond to a series of questions, providing a valuable opportunity for real-time interview practice.

Technologies Used in This Project

  • Frontend HTML, CSS, JavaScript
  • Hosting Vercel

Key Features

Simulation Duration Selection

5, 10, 15 minutes duration options available for the simulation.

Tailored Questions

Questions are customized based on the user's work experience.

Clean Interface

A simple, beginner-friendly interface for ease of use.

User Guide

Instructions for using the application are provided via a guide located in the bottom-left corner.
