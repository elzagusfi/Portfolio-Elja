Project Vending Machine Sales Analysis Dashboard

This project started with my interest in understanding how data can influence business decision-making. I analyzed consumer purchasing data using Python and SQL, then visualized the results with Looker. The resulting dashboard was then implemented on a website to provide easier access for users.

Technologies Used in This Project

  • Data Analysis Python, SQL (BigQuery)
  • Web Dashboard HTML, CSS, JavaScript
  • Visualization & Tools Looker Studio
  • Deployment Vercel

Key Features

Interactive Dashboard

View sales graphs and insights in an interactive and user-friendly dashboard.

Dynamic Data Table

Browse through dynamic data tables based on location and category filters.

Recommendations

Get product recommendations based on location and sales data.

Navigation Menu

Easily navigate through products, locations, active machines, and favorite products.
